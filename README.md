### 📊 Análise de Faturamento de E-commerce com Python
## Objetivo

Analisar dados de vendas de um e-commerce, gerando insights de negócio. Inicialmente no Colab e depois no VS Code com dados limpos e confiáveis.

### Tecnologias

Python, Pandas, Matplotlib
Excel, PostgreSQL
VS Code
IA para interpretação de resultados

### Métricas Principais

Faturamento total: R$ 3.691.820,42

Total de vendas: 337

Receita por canal: Marketplace > Site > Loja Física

### Insights
Marketplace e Site são os canais mais fortes.
Picos de faturamento em Setembro e Novembro.
Março com valores baixos indica mês incompleto ou comportamento atípico.
